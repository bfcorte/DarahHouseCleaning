# Darah Cleaning Services — Static Landing

Modern static site (HTML + Tailwind via CDN). No build tools needed.

## Quick Start
Open `index.html` in a browser.

## Set your WhatsApp
Edit `index.html` and replace the `wa.me/14075551234` link with your number (with country code).

## Publish on GitHub Pages
1. Create a new repo on GitHub (e.g., `darah-cleaning`).
2. Upload all files from this folder (including `index.html` and `assets/`).
3. Go to **Settings → Pages**.
4. In **Build and deployment**, choose **Deploy from a branch**.
5. Select **Branch: main** and **/ (root)**. Save.
6. After it builds, your site will be live at `https://<seu-usuario>.github.io/darah-cleaning/`.

## Publish on Vercel
1. Push this folder to a new GitHub repo.
2. On Vercel: **New Project → Import Git Repository**.
3. Framework preset: **Other** (Static Site). Build command: **None**. Output directory: **/**.
4. Deploy.

## Assets
- `assets/logo.svg` — vector logo (DR monogram + roof + sparkle)
- `assets/favicon.png` — 32×32 favicon
- `assets/icons.svg` — SVG sprite (also inlined inside `index.html`)
